Arduino学习

November 11, 2018

Arduino

  Arduino是一个开放源码电子原型平台，拥有灵活、易用的硬件和软件。Arduino可以接收来自各种传感器的输入信号从而检测出运行环境，并通过控制光源，电机以及其他驱动器来影响 其周围环境。板上的微控制器编程使用Arduino编程语言（基于Wiring）和Arduino开发环境（以Processing为基 础）。Arduino可以独立运行，也可以与计算机上运行的软件（例如，Flash，Processing，MaxMSP）进行通 信。Arduino开发 IDE 接口基于开放源代码，可以让您免费下载使用开发出更多令人惊艳的互动作品。

Arduino UNO

  数字扣和模拟扣，即I/O。数字口有0~13，模拟口有0~5。
  UNO可以通过两种方式供电方式，一种通过USB供电，另一种是通过外接 6~12V的DC电源。ON是电源指示灯，通电就会亮了。L是接在数字口13上的一个LED，在下面一节会有个样例来说明的。TX、RX是串口通讯指示灯，比如我们在下载程序的过程中，这两个灯就会不停闪烁。

安装

  1.下载Arduino ZIP解压文件夹。
  2.安装驱动：Arduino UNO和电脑相连，打开ON指示灯。打开控制面板，选择设备管理器。找到其他设备>Arduino-XX，右击选择更新驱动程序软件。
  3.在弹出的对话框中选择下面一项–> 手动查找并安装驱动程序软件。
  4.打开到Arduino IDE安装位置，就是上面那个解压文件的位置，选择搜索路径到drivers，点击下一步。
  5.选择始终安装此驱动程序软件，直至完成。
  

下载一个Blink程序

  下载一个最简单的代码，既可以帮你熟悉如何下载程序，同时也测试下板子好坏。UNO板上标有L的LED。这段测试代码就是让这个LED灯闪烁。插上USB线，打开Arduino IDE后，找到“Blink”代码。

  点击“校验”。在以后写代码的过程中，输入完代码，都需要校验一下，然后再下载到Arduino中。
  在下载程序之前，我们还要先告诉Arduino IDE板子型号以及相应的串口。选择所用的板卡Board → Arduino UNO。 选择当前的串口——COM口。最后，点击“下载”。

以后程序下载就照着这个步骤做就可以了，再理一下思路，分为三步走: 校验 → 选择boards和com → 下载！

如何加载库

1.先把库文件从网站上下载下来，整个压缩包解压到 Arduino IDE 的libraries文件夹中。需要注意的是，库文件夹下要直接显示.cpp和.h文件，绝对不可以把这些库文件再套到二级以上目录，这样子就会导致IDE无法识别。
2.接下来直接用 Arduino IDE 运行Example里面的程序就可以了，注意不同代码对应的引脚不同。在代码中改为你连接的引脚就行了。

4.呼吸灯

  • void fadeOn(unsigned int time,int increament){
     for (byte value = 0 ; value <255; value+=increament){
      analogWrite(ledPin, value);
      delay(time/(255⁄5));
      }
    }
    fadeOn()、fadeOff()
  • analogWrite(ledPin, value)。该函数作用是发送一个模拟值到一个数字引脚。使用这个函数是要具备特定条件的——该数字引脚需具有PWM功能。观察一下Arduino板，查看数字引脚，你会发现其中6个引脚（3、5、6、9、10、11）旁标有“~”，这些引脚不同于其他引脚，因为它们可以输出PWM信号。value是0~255的数字。
  • PWM是一项通过数字方法来获得模拟量的技术。数字控制来形成一个方波，方波信号只有开关两种状态（也就是我们数字引脚的高低）。通过控制开与关所持续时间的比值就能模拟到一个0到5V之间变化的电压。开（学术上称为高电平）所占用的时间就叫做脉冲宽度，所以PWM也叫做脉冲宽度调制。
    duty cycle叫占空比，每个analogWrite(value)中写入的value都能对应一个占空比，它指的是一个周期内高电平持续时间比上低电平持续时间得到的百分比。。图中，从上往下，第一个方波，占空比为0%，对应的value为0。LED亮度最低，也就是灭的状态。高电平持续时间越长，也就越亮。所以，最后一个占空比为100%的对应value是255，LED最亮。50%就是最亮的一半了，25%则相对更暗。

5.炫彩RGB LED

RGB分为共阳和共阴两种，区别有两点：
（1）接线中的改变，共阳的话，共用端需要接5V，而不是GND，否则LED不能被点亮；
（2）第二点就是，在颜色的调配上，与共阴是完全相反的。 举个例子：共阴RGB显示红色为R-255，G-0，B-0。然而共阳则完全相反，RGB数值是R-0, G-255, B-255。

6.报警器

意蜂鸣器长脚为正（+），短脚为（-）。其原理是利用正弦波产生不同频率的声音。
1.压电式和电磁式：压电式蜂鸣器需要比较高的电压才能有足够的音压,一般建议为9V以上。电磁式蜂鸣器用1.5V就可以发出85dB以上的音压了，唯消耗电流会大大的高于压电式蜂鸣器；
2.有源和无源：“源”不是指电源，而是指振荡源，有源蜂鸣器（有长短脚）内部带振荡源，说白了就是只要一通电就会响。而无源（无长短脚）内部不带震荡源，所以如果仅用直流信号无法使其响，必须用2K-5K的方波去驱动它。
* sinVal=(sin(x*(3.1412⁄180)));将角度转化为弧度 * toneVal =2000+(int(sinVal*1000));现在toneVal就是一个适合声音频率了。
* tone()函数：
 1.tone(pin,frequency):frequency是以Hz为单位的频率值，本案例中为toneVal
 2.tone(pin,frequency,duration):duration参数是以毫秒为单位，表示声音长度的参数。如果没有指定duration，声音将一直持续直到输出一个不同频率的声音产生。
 3.noTone(pin):结束该指定引脚上产生的声音

7.温度报警器

  • LM35温度传感器：把温度信号变为电压的改变。Arduino的模拟引脚连接到一个了10位A/D转换， 输入0 ~ 5V的电压对应读到0~1023的数值，每个读到的温度数值对应的都是一个电压值。每10mV对应1摄氏度。

    对于这个传感器：Vout = 10mV/℃ * T℃
    Datasheet会提供出场芯片的性能参数及一些简单的电路搭建。
  • 成功下载完程序后，打开Arduino IDE的串口监视器。设置串口监视器的波特率为9600。就可以直接从串口中读取温度值，并尝试升高周围环境温度，或者用手直接接触LM35使其升温，串口可以很直观的看到温度有明显的变化。
  • 串口：Arduino与外界进行通信的一个简单办法。UNO分别和数字引脚0（RX）和数字引脚1（TX）相连。所以两个引脚不能做输入输出功能，也不能在下载程序时接东西。
  • Val=analogRead(Pin)从某模拟口读值
  • data = (double) val *(5⁄10.24);从传感器中读到的电压值，它的范围在0~1023，将该值分成1024份， 再把结果乘以5 ， 映射到0~5V，因为每度10mV，需要再乘以100得到一个double型温度值，最后赋给data变量。

  • serial.begin(9600):初始化串口的波特率，也就是数据传输的速度 serial.print(val):
    （1）数字则是以位形式输出;
    （2）浮点型数据输出时只保留小数点后两位;
    （3）字符和字符串则原样输出，字符需要加单引号，字符串需要加双引号。
    serial.printIn(val):println()比print()多了回车换行，其他完全相同。

    8.震动传感器

  • 滚珠开关：需要一个下拉电阻。通过珠子滚动接触导针的原理来控制电路的通断。金色导针一端是导通的，银色导针一端是不导通的。

  • 中断函数attachInterrupt(interrupt,function,mode)是一个当外部发生中断时，才被唤醒的函数。区别于其他函数，它依附于中断引脚才发生。大多数板子都有两个外部中断引脚： 数字引脚2（ 中断0 ） 和数字引脚3 （ 中断1）中断0与中断1是中断号，在函数中需要用到。不同板子，中断号对应引脚可能不同，可以查阅Arduino官方编程语法手册
    interrupt：中断号0或者1。如果选 低电平时，触发中断。择0的话，连接到数字引脚2上，选择1的话，连接到数字引脚3上；
    function：调用的中断函数名。写中断函数时，需要特别说明：
    1、我们在写中断函数的时候，该函数不能含有参数和返回值。也就是说，要是一个无返回值的函数；
    2、中断函数中不要使用delay()和millis()函数，因为数值不会继续变 化；
    3、中断函数中不要读取串口，串口收到的数据可能会丢失。
    mode：中断的条件。只有特定的以下四种情况：
    1、LOW 当引脚为低电平时，触发中断；
    2、CHANGE 当引脚电平发生改变时，触发中断；
    3、RISING 当引脚由低电平变为高电平时，触发中断；
    4、FALLING 当引脚由高电平变为低电平时，出发断电。

10.舵机初动

  • 舵机是一种电机，用一个反馈系统控制其位置，较多用于对角度有要求的场合，如摄像头、机器人关节；
  • #include ——声明调用Servo.h库，库已存在libraries/Servo/Servo.h。库就是将多个函数封装打包。函数名是函数的标签，库名是也叫“对象”。Servo myservo就是创建一个叫myservo的舵机对象，myservo.attach(pin)就是调用Servo库中的attach函数，即将引脚上的舵机与声明的舵机对象连接起来，pin不建议使用数字0，1。
  • 电位器是一个可变电阻，连通模拟引脚，依靠电压（0~1023）转换成角度（0~179），所以要用map函数，map(value，fromLow, fromHigh, toLow, toHigh)，将一个数从一个范围映射到另外一个范围。
