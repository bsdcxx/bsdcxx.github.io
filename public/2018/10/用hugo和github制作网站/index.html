用hugo和github制作网站

October 30, 2018

安装git

1.安装git .如果你不确定自己的电脑是否安装了Git或者安装之后不知道安装是否完成，都可以使用git的版本查询完成确认：git –version

2.安装了Git，为了保证Git和Github能无密码链接，也就是为后续工作减轻输入密码的工作量，可以使用SSH的方式建立连接，如果你觉得每次输入用户名密码不碍事，也可以忽略这个操作。 输入：ssh-keygen -t rsa -C “email address”

使用这个命令 ssh-keygen 一般会要求确认位置和输入密码，这写操作都使用回车就能解决。
用下列语句提取ssh代码：cat ~/.ssh/id_rsa.pub

复制提取的全部信息，粘贴到 Github SSH Keys管理页面 上。起一个好记忆的名字一保存就完成了SSH链接的工作。

3.建立仓库 git init

在Github上，选择“New Repository”链接，起一个名字，点击”Create repository“按钮就完成了相关的建仓库操作。
4.在init后的文件下，运行命令：git add . 就可以把所有文件都添加到我们新建的本地仓库中了。这个操作很简单，也就完成了添加文件的操作。
5.提交内容git commit -m “text”
6.git remote add origin https://github.com/[user]/[初始文件名].git
7.推送文件git push -u origin master。完成推送，Github上就能看到我们上传的文件了。在第一次完成上述步骤之后，以后的网站更新就只需要完成第4、5、7步所涉及的步骤。
